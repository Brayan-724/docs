## Installation

<CodeGroup>

```bash Linux/MacOS
curl -fsSL http://densky.apika.me/install.sh | sh -s v0.1.0-alpha1
```

```bash Windows
# For Windows
irm https://densky.apika.me/install.ps1 | iex
```

```bash Cargo
cargo install densky
```

</CodeGroup>

## Create a new project
Start a new Densky project:
```bash
densky init <folder-name>
```

 Run the dev command to start Densky in development mode:
```bash
densky dev
```
